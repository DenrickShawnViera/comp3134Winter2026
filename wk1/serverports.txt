1. What is the definition of a server port?
A server port is a numerical identifier used by a computer to determine which application or service should receive incoming network traffic.

2. What is server port range?
Server ports range from 0 to 65535.

3. How many bits does the number above represent?
The port number range is represented using 16 bits.

4. State 10 common server ports:
1) 21 – FTP
2) 22 – SSH
3) 23 – Telnet
4) 25 – SMTP
5) 53 – DNS
6) 80 – HTTP
7) 110 – POP3
8) 143 – IMAP
9) 443 – HTTPS
10) 3306 – MySQL
